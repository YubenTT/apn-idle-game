# Concept

## Why this game exists

All Patch Notes is where gamers wait for **patch notes, news, events, videos**.  
**APN Idle** turns that wait into a playful loop that teaches the brand language of the site.

## Fantasy

You are the **APN signal scout** — Host mascot on the Signal Road.

- **Noise** (stale posts, rumor bots, lag spikes) blocks the feed.  
- **Patch Notes** (crimson) carry real **Notes**.  
- You **Publish** them for **Rep** and grow **Live Mult** across seasons.

## Mapping to the site

| Site | Game |
|------|------|
| Patch notes | Notes currency + red Patch Note enemies |
| News / Video / Event / Guide | Type chips + enemy flavors + LIVE ticker |
| Verified / summary / tracker | Crit Mask, Area Blast, Damage Ramp |
| Live feed | Live Mult prestige |
| Tools / scanner | Upgrade Weapon |
| Hotfix energy | Burst Hit skill |

## Loop

```text
March → clear noise → Signal → Upgrade Weapon
     → rank SP → Damage / Crit / Skills
     → Notes → Publish → Rep → Boosts
     → Version Gates → every 20 zones checkpoint → Live Mult
     → zones continue forever
```

## Production embed

Static folder under `apn-web/public/idle/` or `/play` iframe.  
No backend required for v1 (`localStorage`).

## Satisfaction lessons (PlayForge-class idle)

| Pillar | APN Idle |

| Cast personality | Host + feed-noise enemies |
